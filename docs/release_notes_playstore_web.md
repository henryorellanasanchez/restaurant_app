# Release Notes - v1.0.1+2

Fecha: 2026-04-25
Aplicación: La Peña Bar & Restaurant

## 1. Texto para Play Store

### Título corto
Mejoras de estabilidad, sincronización y experiencia móvil

### Novedades de esta versión
- Mejoramos la estabilidad general para operación en producción.
- Refinamos sincronización y estado de nube para mayor claridad.
- Pulimos navegación y accesibilidad visual en móvil y tablet.
- Mejoramos la experiencia en módulos clave: Mesas, Pedidos, Cocina, Caja, Reservas, Usuarios, Reportes y Dashboard.
- Optimizamos la versión web pública para mejor respuesta en diferentes tamaños de pantalla.
- Ajustamos compatibilidad Android para empaquetado release con Firebase Firestore.

### Nota técnica opcional (uso interno)
Build validado con AAB y APK release, más build web release.
